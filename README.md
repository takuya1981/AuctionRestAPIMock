# AuctionRestAPIMock
REST APIのモック

参照：  
　Jersey  
　Jetty  
　glassfish  
　postgresql  

公開しているAPI：  
  オークションアイテム出品　POST AuctionItem  
  オークションアイテムリスト取得　GET GetAuctionList  
  オークションアイテム詳細取得　GET GetAuctionItemDetail  
  オークション入札　PUT AuctionBid
